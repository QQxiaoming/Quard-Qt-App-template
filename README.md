# Quard的Qt应用程序初始模板

本模板是为了方便Quard（本项目作者）创建新的Qt应用程序项目而设计的。它包含了基本的无边框窗口、左侧边页、主页面、底部终端、右侧通知中心，亮/暗色主题切换，顶部与标题栏结合的菜单栏等基本功能。本模板支持跨平台，可以在Windows、Linux、MacOS等系统上运行。需要Qt 6.5.0及以上版本。
